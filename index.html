Introduction

Our Team

This project was created for the university course Tools of Software Projects.
Our team consisted of 3 people:

  • Balazs Katona - Makefile, Sounds and Music
  • Daniel Eke - Testing, Menus, helped with GameLogic
  • Benjamin Ferenc Hajas - GameLogic, GameScene, Documentation, helped with Menus

The Idea

Our goal was to create a two-dimensional game very similar to an existing one.

The original game

The game can be found at http://phoboslab.org/ztype/ .

The game

The game itself is an infinite shooter.
Each wave of enemies is stronger than the previous one.
After a while the player is not able to kill every enemy on screen so they lose.

On every enemy there is a random English word from the official Scrabbler dictionary. If the player can type in the word before an enemy gets to them, the enemy dies and the player gets points.

Realization

We decided to make this game in C++.
Since Daniel worked on Macintosh and Me and Balazs both worked on Windows, we needed a cross platform library.
We decided we would use SFML because it is very well documented and simple to use.


We used GTest for the tests, we use a makefile to build and Doxygen for the documentation.
